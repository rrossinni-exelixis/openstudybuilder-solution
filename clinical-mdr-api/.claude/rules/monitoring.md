# Telemetry & Monitoring

- **Tracing**: OpenCensus with Azure Application Insights or Zipkin support
- **Logging**: Structured logging via `common/logger.py`
- **Metrics**: Request metrics via `common/telemetry/request_metrics.py`
- **Configuration**: Set `TRACING_ENABLED=true` and `APPLICATIONINSIGHTS_CONNECTION_STRING` in `.env`
